# SAFS



## Installation
```
git clone https://github.com/automl/SAFS.git
cd SAFS
conda create -n safs python=3.8
conda activate safs

# Install for usage
pip install .

# Install for development
make install-dev
```

Documentation at https://automl.github.io/SAFS/main
